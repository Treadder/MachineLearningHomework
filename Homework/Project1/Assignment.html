Important Note:


There are several ways to start Jupyter, but we need to do so in a way that it will be able to access pandas and other important python modules. These have been installed with Anaconda, so the simplest way to do this is:

- open up your .bashrc file in your home directory, and insert this line in the file:

PATH="/opt/anaconda3/bin:$PATH"


Then try:

- opening a terminal,

- 'cd' to a directory that you want to have your notebook(s) in,

- enter 'jupyter notebook' at the command line.

This should open a browser, at which point you can create a new notebook (look in the upper right). Then type 'import pandas' in a cell and run it - it should now work.
